# Auteo 品牌标志

## 用途

`assets/logo.svg` 是 Auteo 的通用矢量品牌标志，用于界面、文档和其他需要透明背景标志的场景。macOS 应用图标继续使用 `assets/app-icon.png`，不由该文件替代。

## 视觉规范

- 标志只包含两个几何图形，不包含背景、渐变、描边、阴影或辉光。
- 图形通过中间的负空间形成播放符号，并保留原应用图标中的圆润外轮廓和斜向结构。
- SVG 使用方形视口并在四周保留安全空间，缩放时不得裁切或改变宽高比例。
- 标志不定义固定品牌色，填充色使用 `currentColor`，由宿主界面决定最终颜色。

## 使用规则

- 在支持内联 SVG 的界面中，通过 CSS `color` 属性设置标志颜色。
- 作为独立图片加载时，使用 SVG 的默认当前颜色；需要动态换色的场景应优先内联 SVG。
- 不得在 SVG 内加入固定色值，以确保浅色、深色和高对比度界面都能复用。
- 必须提供可访问名称；当前文件通过 `title` 和 `aria-labelledby` 声明名称。

## 限制与非目标

- 该文件不是 macOS 应用图标源文件，不负责应用图标所需的背景和视觉特效。
- 该文件不包含文字标识，也不定义标志与产品名称组合时的排版规则。
- 该文件不提供多色或渐变版本。
